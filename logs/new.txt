## PROJECT OVERVIEW
===================

**Project Name:** KITS Alumni Association Website
**Technology Stack:** Next.js 15.3.1, React 19, TypeScript, Tailwind CSS, Supabase
**Purpose:** Alumni network platform for KKR & KSR Institute of Technology and Sciences (AI & ML Department)
**Color Scheme:**
- Primary: #3F426B (Burgundy)
- Secondary: #D4A72E (Gold)
- Text: #FFFFFF (White background)

### 6. DATABASE ARCHITECTURE
---------------------------

**Supabase Configuration:**
- Real-time database with PostgreSQL
- Row Level Security (RLS) enabled
- Custom functions and procedures

**Tables:**

**admins**
- id (string, primary key)
- email (string, unique)
- created_at, updated_at (timestamps)
- Password management through custom functions

**blogs**
- id (string, primary key)
- name (string, author name)
- title (string, blog title)
- blog (text, blog content)
- status (string, pending/approved/rejected)
- created_at, updated_at (timestamps)

**health_check**
- status (string, system health indicator)

**Custom Functions:**
- exec_sql: Execute raw SQL queries
- add_admin: Add new admin users
- add_student_blog: Add student blog posts
- get_blogs_as_json: Retrieve blogs as JSON
- get_student_blogs: Get blogs by student name
